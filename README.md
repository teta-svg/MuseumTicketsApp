# Курсовой проект: Web-приложение для продажи билетов в музейные комплексы

**Тема курсовой:** Разработка Web-приложения для управления музейными комплексами, выставками и продажей билетов онлайн.

## О проекте

Это учебный проект, демонстрирующий работу с базой данных и C# через Entity Framework Core для создания веб-приложения

## Состав проекта

- **Скрипт для базы данных**: `CreateDatabase.sql`  
  Создает базу данных `MuseumTicketsDB` и все таблицы (Museums, Exhibitions, Tickets, Orders и т.д.).
  
- **Модели C# и DbContext**: `Models/MuseumContext.cs` и остальные модели EF Core  
  Сгенерированы через **Database First** (`Scaffold-DbContext`) для работы с базой данных.
