# Курсовой проект: Web-приложение для продажи билетов в музейные комплексы

**Тема курсовой:** Разработка Web-приложения для управления музейными комплексами, выставками и продажей билетов онлайн.

## О проекте

Проект построен на базе **.NET 10 / C#** с использованием **Entity Framework Core** и принципов многослойной архитектуры (**Clean Architecture**).

---

## Структура решения
Решение разделено на логические слои для обеспечения чистоты кода:

### 1.  Core (Ядро)
*   **Museum.Domain** — Содержит сущности предметной области
*   **Museum.Application** — Логика приложения. Содержит:
    *   **DTOs**
    *   **Interfaces**
    *   **Services**

### 2. Infrastructure (Инфраструктура)
*   **Museum.Persistence** — Слой доступа к данным. 
    *   Содержит `MuseumTicketsDBContext`.
    *   Реализовано разделение конфигураций: настройки таблиц вынесены из контекста в папку `Configurations` (использование Fluent API)

### 3. Presentation (Представление)
*   **Museum.WebAPI** — Точка входа в приложение.

---

## Технологический стек
*   **Язык:** C# / .NET 10
*   **ORM:** Entity Framework Core (Database First)
*   **СУБД:** Microsoft SQL Server (LocalDB)

---

## Быстрый старт (дорабатывается)
1.  **База данных:** Выполнить SQL-скрипт `CreateDatabase.sql` для создания таблиц.
2.  **Запуск:** 
    *   `Museum.WebAPI` является запускаемым проектом.
    *   Запустить API. Документация методов доступна через **Swagger**.
