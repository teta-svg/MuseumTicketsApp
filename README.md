# Курсовой проект: Web-приложение для продажи билетов в музейные комплексы

## О проекте

Проект разработан на базе **.NET 10 / C#** с использованием **Entity Framework Core**.

Система предназначена для управления выставками, продажей билетов, обработкой заказов и взаимодействием с пользователями.

---

## Технологический стек

* **Язык:** C# / .NET 10
* **Frontend:** Blazor (Interactive Server)
* **JS Interop:** LocalStorage, File Download
* **ORM:** Entity Framework Core
* **СУБД:** Microsoft SQL Server
* **Безопасность:** JWT (JSON Web Token)

---

## Взаимодействие с JavaScript

В проекте используется минимальный JS Interop:

* **Аутентификация (`window.auth`)**

  * хранение JWT-токена в `localStorage`
  * управление сессией пользователя

* **Работа с файлами (`window.saveAsFile`)**

  * скачивание файлов (билеты, отчёты)
  * передача данных через Base64

---

## Быстрый старт

1. Установить проект **Museum.BlazorUI** как стартовый
2. Запустить приложение
3. Перейти в браузере:

   * UI: `/`
   * Swagger: `/swagger`

## Возможности системы

* Просмотр выставок
* Фильтрация и поиск
* Покупка билетов
* Управление заказами
* Регистрация и авторизация
* Администрирование:

  * управление выставками
  * управление пользователями
  * генерация отчётов (Excel)
